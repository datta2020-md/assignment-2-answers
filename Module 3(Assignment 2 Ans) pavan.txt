2.   Write a python program to count the number of charters (character frequency in string). 
Ans: def char_frequency(str1):
     dict = {}
     for n in str1:
     keys = dict.keys()
     if n in keys:
     dict[n] += 1
     else:
     dict[n] = 1
     return dict
     print(char_frequency('google.com'))
